# Vue.js Developer Dashboard

A modern Single Page Application (SPA) built with Vue.js and Vite to
demonstrate the fundamentals of frontend development using a
component-based architecture.

This project features a professional developer dashboard interface with
multiple pages, reusable components, reactive state management, project
filtering, and client-side routing. It serves as an educational project
for learning core Vue.js concepts while building a responsive and visually
appealing user interface.

## Features

* Modern dashboard user interface
* Vue Router navigation
* Reusable Vue components
* Reactive state management with `ref()`
* Computed properties with `computed()`
* Dynamic list rendering with `v-for`
* Two-way data binding with `v-model`
* Custom events with `defineEmits()`
* Component communication using Props and Events
* Project filtering functionality
* Responsive design
* Custom CSS styling and animations

## Technologies Used

* Vue.js 3
* Vite
* Vue Router
* JavaScript (ES6+)
* HTML5
* CSS3

## Learning Objectives

This project helps students understand:

* Vue project structure
* Component-based architecture
* Reactive programming
* State management fundamentals
* Client-side routing
* Parent and child component communication
* Dynamic rendering techniques
* Modern UI development practices
* SPA application architecture
